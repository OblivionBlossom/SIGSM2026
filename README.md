# S.I.G.S.M. - Sistema Informático de Gestión de Servicios Médicos

## Tecnologías utilizadas
* HTML
* MySQL

## Pasos para la Instalación
Sigue estos pasos para tener el proyecto en tu computadora:
1. Haz clic en el botón verde que dice **"Code"** aquí arriba en GitHub.
2. Selecciona **"Download ZIP"** y descárgalo.
3. Descomprime el archivo `.zip` en la carpeta que prefieras de tu computadora.
4. Asegúrate de tener instalado XAMPP 8.x en tu computadora.
5. Copia la carpeta descomprimida dentro de la carpeta **`htdocs`** de XAMPP.

## Pasos para el Uso
Para ver la página web en funcionamiento:
1. Abre el panel de control de XAMPP e iniciá los servicios **Apache** y **MySQL**.
2. Abre tu navegador web.
3. Escribe en la barra de direcciones: `http://localhost/nombre-de-la-carpeta`
